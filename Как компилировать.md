# Сборка проекта


#### Этап А: Конфигурация проекта

Выполняется при первом запуске или если вы добавили в проект **новые файлы `.cpp`**:

```bash
cmake -B build -G "MinGW Makefiles"

```

*Эта команда создаст папку `build` и сгенерирует файлы для сборщика.*

#### Этап Б: Компиляция кода

Выполняется каждый раз, когда вы **изменили код** в существующих файлах:

```bash
cmake --build build

```

*Компилятор выполнит инкрементальную сборку, обновив только измененные файлы.*

---

### Запуск программы

После успешного выполнения этапа компиляции готовый исполняемый файл появится в папке `build`. Его можно открыть через проводник или через терминал:

```bash
.\build\token_counter.exe

```
