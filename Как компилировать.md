## 🚀 Быстрый старт для команды (Windows + MSYS2)

Поскольку проект разрабатывается под чистой Windows, мы используем изолированное окружение **MSYS2 (UCRT64)**. Это избавляет от необходимости вручную прописывать системные переменные `PATH` и настраивать компиляторы в самой Windows.

1. Откройте терминал **MSYS2 UCRT64** (синий ярлык в меню «Пуск»).
2. Обновите базу пакетов и установите весь необходимый тулчейн (компилятор, CMake, Make и библиотеку RE2) одной командой:

### 2. Переход в папку репозитория

### 3. Сборка проекта

Сборка проекта выполняется стандартными командами CMake в два этапа:

#### Этап А: Конфигурация проекта

Выполняется при первом запуске или если вы добавили в проект **новые файлы `.cpp**`:

```bash
cmake -B build -G "MinGW Makefiles"

```

*Эта команда создаст папку `build` и сгенерирует файлы для сборщика.*

#### Этап Б: Компиляция кода

Выполняется каждый раз, когда вы **изменили код** в существующих файлах:

```bash
cmake --build build

```

*Компилятор выполнит инкрементальную сборку, обновив только измененные файлы.*

---

### 4. Запуск программы

После успешного выполнения этапа компиляции готовый исполняемый файл появится в папке `build`. Запустить его можно прямо из терминала MSYS2:

```bash
./build/token_counter.exe

```
