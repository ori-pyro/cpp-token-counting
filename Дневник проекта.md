## Постановка задачи:
- Необходимо подсчитать количество лексем в двух проектах на C++. Результат должен быть представлен в виде таблицы: для каждого вида лексем представлено количество их применения в коде проекта.

---

## Описание алгоритма:
1. На вход подаётся путь к папке на локальном диске.
2. Программа проверяет каждую папку и ищет файлы с расширением `.cpp`, `.h`
	- Определение лексем происходит при помощи регулярных выражений.
3. После проверки каждого файла вносятся изменения в таблицу в формате  `.json/.md`
